# Prime Habits

A simple Habitify-style habit tracker web app.

## Features

- Add and delete habits
- Mark habits complete for today
- Daily completion percentage
- Streak counter
- Weekly progress bars
- Daily reflection box
- ChatGPT prompt generator
- Saves data in browser using localStorage

## How to run

Open `index.html` in your browser.

## How to deploy on Vercel

1. Go to Vercel.
2. Create a new project.
3. Upload this folder.
4. Since this is static HTML/CSS/JS, no build command is needed.

## Files

- `index.html` = app layout
- `styles.css` = design
- `app.js` = app logic
